\begin{frame}{Computadores}

A melhor alocação dos alunos poderia ser obtida distribuindo de maneira igualitária entre os horários. No nosso exemplo, temos 8 alunos e cada
aluno precisa estudar duas horas seguidas. Logo, temos no total 16 horas (TotalHoras) que precisam ser distribuídas. Temos 6 horários disponíveis (NumHorarios). Então,
o menor número de computadores (Computadores) necessários é igual:

\pause
\begin{equation}
Computadores = \displaystyle \Bigl \lceil \dfrac{TotalHoras}{NumHorarios} \Bigr \rceil 
\end{equation}

\pause
No nosso exemplo, 

\begin{equation}
Computadores = \displaystyle \Bigl \lceil \dfrac{16}{6} \Bigr \rceil  = \Bigl \lceil 2.66 \Bigr \rceil = 3
\end{equation}

\pause
Observe que este é um limite inferior para o número de computadores utilizado. Observe que pode não existir uma distribuição que utilize esse número de computadores.
Mas se uma distribuição utiliza essa quantidade de computadores, ela será a melhor possível.



\end{frame}

\begin{frame}{Distância entre palavras}
Vamos definir três tipos de operações básicas:
\begin{itemize}
\item inserir uma letra em uma palavra;
\item remover uma letra de uma palavra;
\item substituir um letra de uma palavra.
\end{itemize}

Definimos também a distância entre duas palavras como o número mínimo de operações básicas
para transformar a primeira palavra na segunda. Por exemplo, a distância entre as palavras maria
e clara é 3:
\begin{enumerate}
\item remover a letra i: maria $\to$ mara
\item substituir a letra m pela letra c: mara $\to$ cara
\item inserir a letra l: cara $\to$ clara 

\end{enumerate}

\end{frame}

\begin{frame}{Mesas - Resolução}
Para calcular a quantidade de mesas, podemos utilizar a fórmula que encontramos:
\begin{equation*}
    qntCadeiras = 2 + (4\ast qntMesas)
\end{equation*}
Dessa vez, vamos substituir a quantidade de cadeiras (qntCadeiras) e encontrar a quantidade de mesas (qntMesas). Então, para 42 estudantes temos que
\begin{equation*}
    42 = 2 + (4\ast qntMesas)
\end{equation*}
Mudando de lado na equação, temos que
\begin{equation*}
    2 + (4\ast qntMesas) = 42
\end{equation*}
Fazendo a manipulação algébrica
\begin{equation*}
4\ast qntMesas = 42 - 2
\end{equation*}
\begin{equation*}
4\ast qntMesas = 40
\end{equation*}
Logo, 
\begin{equation*}
 qntMesas = 40\div4 = 10 mesas
\end{equation*}
