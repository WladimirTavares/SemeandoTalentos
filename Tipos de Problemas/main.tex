\begin{frame}{Tipos de Problemas}

\begin{itemize}
 \pause \item \textbf{Problemas Lógicos:} Problemas que existem um raciocínio lógico apurado e uma boa análise de casos. Podem ser divididos em: 
 \begin{itemize} 
 \pause \item \textbf{Problemas de Agrupamento}: Construção de um ou mais grupos satisfazendo regras.
 \pause \item \textbf{Problemas de Ordenação}: Construção de ordenação de objetos satisfazendo regras.
 \end{itemize}
 \pause \item \textbf{Problemas de Análise}: Encontrar um número de operações de um certo algoritmo sobre um conjunto de dados.
 \pause \item \textbf{Problemas de Simulação}: Tarefas simples que pedem aos competidores que sigam um conjunto bem definido de
instruções.
 \pause \item  \textbf{Problemas Algorítmicos}: Tarefas não pedem explicitamente um algoritmo, mas os estudantes são encourajados a desenvolver um método sistemático para resolver estas tarefas.
\end{itemize}

 
\end{frame}


\begin{frame}{Estrutura Geral dos Problemas Lógicos}

A estrutura do problema pode ser dividida em três partes:

\begin{itemize}
\item Cenário: introduz conjuntos de variáveis (pessoas, locais,
coisas, eventos etc).
\item Regras: conjunto de declarações que descreve as relações
entre as variáveis.
\item Perguntas: conjunto de perguntas relacionadas às variáveis
e às regras.
\end{itemize}

\end{frame}

\begin{frame}{Problemas de Ordenação}

\begin{itemize}
 \item Neste tipo de problema, precisamos construir uma ordem dos objetos respeitando as regras que podem ser:
 \begin{itemize}
 \pause \item Regras de posicionamento: Objeto X está na posição $i$.
 \pause \item Regras de sequenciamento: Objeto X vem antes (ou depois) de Y  
 \pause \item Regras condicionais: Se Objeto X vem antes (depois) de Objeto Y então Objeto W vem antes (depois) do Objeto Z.
 \pause \item Regras de combinação: O objeto X (não) deve ser adjacente ao Objeto Y 
 \end{itemize}
\
\end{itemize}

 
\end{frame}

\begin{frame}{OBI 2019 NJF2 - Distância entre palavras}
Vamos definir três tipos de operações básicas:
\begin{itemize}
\item inserir uma letra em uma palavra;
\item remover uma letra de uma palavra;
\item substituir um letra de uma palavra.
\end{itemize}

Definimos também a distância entre duas palavras como o número mínimo de operações básicas
para transformar a primeira palavra na segunda. Por exemplo, a distância entre as palavras maria
e clara é 3:
\begin{enumerate}
\item remover a letra i: maria $\to$ mara
\item substituir a letra m pela letra c: mara $\to$ cara
\item inserir a letra l: cara $\to$ clara 

\end{enumerate}

\end{frame}

\begin{figure}

{\small
O primeiro triângulo de Sierpinski possui 0 triângulos brancos.\\
